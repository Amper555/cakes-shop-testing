# Чек-лист тестирования интернет-магазина "Мастерская ТортИк"

**Объект тестирования:**  
[https://buy-cakes-2.herokuapp.com](https://buy-cakes-2.herokuapp.com)

## 1. Главная страница

### 1.1. Общие элементы
- [ ] Корректное отображение названия "Мастерская ТортИк"
- [ ] Отображение заголовка и описания мастерской
- [ ] Корректная работа выпадающего меню "Для кого"
- [ ] Корректная работа выпадающего меню "Повод"
- [ ] Работоспособность кнопки "Подобрать торт"
- [ ] Адаптивность главной страницы (мобильная версия)

### 1.2. Функциональность подбора торта
- [ ] Выбор значения в меню "Для кого"
- [ ] Выбор значения в меню "Повод"
- [ ] Совместная работа двух фильтров ("Для кого" + "Повод")
- [ ] Отображение результатов подбора после нажатия кнопки "Подобрать торт"
- [ ] Корректное поведение при разных комбинациях фильтров
- [ ] Отсутствие ошибок при очистке фильтров или повторном подборе

## 2. Административная панель (/admin)

### 2.1. Доступ к админке
- [ ] Возможность входа в административную панель
- [ ] Отображение списка тортов после успешного входа

### 2.2. Управление тортами
- [ ] Добавление нового торта
- [ ] Редактирование существующего торта
- [ ] Удаление торта
- [ ] Валидация полей при создании и редактировании (название, цена, описание и т.д.)
- [ ] Корректное отображение изменений после редактирования

## 3. API-тестирование
- [ ] GET /api/cakes — получение списка всех тортов
- [ ] POST /api/cakes — создание нового торта
- [ ] PUT /api/cakes/{id} — обновление торта
