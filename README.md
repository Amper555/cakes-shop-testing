# Функциональное и API-тестирование интернет-магазина тортов

**Объект тестирования:**  
[https://buy-cakes-2.herokuapp.com](https://buy-cakes-2.herokuapp.com)

### О проекте
Пет-проект, целью которого является проведение полного цикла функционального тестирования веб-приложения интернет-магазина тортов.  
Проект включает тестирование клиентской части и административной панели.

### Цель проекта
- Выявить дефекты в работе приложения
- Составить качественную тестовую документацию
- Отработать навыки функционального тестирования, тестирования форм, валидации и API
- Научиться правильно оформлять баг-репорты и отчёты о тестировании

### Виды тестирования
- Функциональное тестирование
- Тестирование пользовательского интерфейса (GUI)
- Тестирование форм и полей ввода
- Тестирование валидации данных
- Позитивное и негативное тестирование
- API-тестирование (REST)

### Инструменты
- Chrome DevTools
- Postman — тестирование REST API
- GitHub
- TestRail / Qase (в процессе изучения)

### Что было сделано
- Составлен подробный чек-лист основного функционала
- Разработаны тест-кейсы для ключевых пользовательских сценариев
- Выявлены и оформлены баг-репорты
- Проведено тестирование административной панели (/admin)
- Выполнено тестирование API эндпоинта `/api/cakes`
- Подготовлен отчёт о результатах тестирования
