# Функциональное и API-тестирование интернет-магазина тортов

**Объект тестирования:**  
[https://buy-cakes-2.herokuapp.com](https://buy-cakes-2.herokuapp.com)

### О проекте
Пет-проект, целью которого является проведение полного функционального тестирования клиентской части и административной панели интернет-магазина тортов.

### Цель проекта
- Выявить дефекты в работе веб-приложения
- Составить тестовую документацию
- Отработать навыки функционального тестирования, тестирования форм, валидации данных и API-тестирования

### Виды тестирования
- Функциональное тестирование
- Тестирование GUI
- Тестирование форм и полей ввода
- Тестирование валидации данных
- Позитивное и негативное тестирование
- API-тестирование

### Инструменты
- Chrome DevTools
- Postman (API-тестирование)
- GitHub
- TestRail / Qase (в процессе изучения)

### Что было сделано
- Составлен чек-лист основного функционала
- Написаны тест-кейсы
- Найдены и оформлены баг-репорты
- Проведено тестирование админ-панели
- Выполнено тестирование REST API (`/api/cakes`)
