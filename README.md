# Функциональное тестирование интернет-магазина тортов

**Объект тестирования:**  
https://buy-cakes-2.herokuapp.com

**Цель проекта:**  
Провести функциональное тестирование клиентской части и админ-панели интернет-магазина тортов, выявить дефекты, составить тестовую документацию.

**Виды тестирования:**
- Функциональное тестирование
- Тестирование GUI
- Тестирование форм
- API-тестирование
- Тестирование валидации данных

**Инструменты:**
- Chrome DevTools
- Postman (для API)
- GitHub
