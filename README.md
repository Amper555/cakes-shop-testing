# Функциональное и API-тестирование интернет-магазина тортов

**Объект тестирования:**  
[https://buy-cakes-2.herokuapp.com](https://buy-cakes-2.herokuapp.com)

### О проекте
Пет-проект, направленный на отработку практических навыков ручного тестирования веб-приложения.  
Было проведено комплексное функциональное тестирование клиентской части и административной панели интернет-магазина.

### Цель проекта
- Выявить дефекты в работе веб-приложения
- Составить тестовую документацию
- Отработать навыки создания чек-листов, тест-кейсов и баг-репортов
- Получить опыт тестирования форм, валидации данных и REST API

### Виды тестирования
- Функциональное тестирование
- Тестирование пользовательского интерфейса (GUI)
- Тестирование форм и полей ввода
- Тестирование валидации данных
- Позитивное и негативное тестирование
- API-тестирование (REST)

### Используемые инструменты
- Chrome DevTools
- Postman (для API-тестирования)
- GitHub
- TestRail / Qase (в процессе изучения)

### Что было сделано
- Составлен чек-лист основного функционала приложения
- Разработаны тест-кейсы для ключевых сценариев
- Выявлены и оформлены баг-репорты
- Проведено тестирование административной панели
- Выполнено тестирование REST API эндпоинта `/api/cakes`
- Подготовлен отчёт о проведённом тестировании
